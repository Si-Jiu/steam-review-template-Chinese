# Steam 評測模板

[简体中文](./README.md) | **正體中文** | [English](./README_EN.md)

這個工具可以簡化建立在Steam測評上流行的複製貼上式評測。選項來源於各種Steam評測，做了小修改。

## 使用

選選項就行了，例如：

```plaintext
---{ 配置要求 }---
☐ 能玩掃雷就行
☐ 土豆
☐ 中端
☐ 中高階
☑ 神機
☐ 神威·太湖之光
```
